# Python Text Editor

Editor de texto simples desenvolvido em Python utilizando a biblioteca Tkinter. Este é um projeto pessoal criado com o objetivo de aprendizado, prática de interfaces gráficas (GUI) e evolução contínua do código. O projeto serve como base para experimentar melhorias, aplicar boas práticas e construir gradualmente um aplicativo mais completo.

## Funcionalidades atuais

- Editor de texto em tela cheia
- Menu superior com opções para salvar arquivos `.txt`, alterar a fonte entre Helvetica e Courier e encerrar o aplicativo
- Interface simples e funcional
- Utiliza apenas bibliotecas padrão do Python

## Tecnologias utilizadas

- Python 3
- Tkinter

## Como executar o projeto

1. Verifique se o Python 3 está instalado executando o comando `python --version`
2. Clone o repositório ou faça o download dos arquivos
3. Execute o arquivo principal com o comando `python texteditor.py`

## Objetivo do projeto

O objetivo deste projeto é aprender e consolidar conceitos de desenvolvimento de interfaces gráficas com Tkinter, praticar organização de código e separação entre interface e lógica, criar uma base sólida para futuras evoluções e evoluir de um editor simples para um aplicativo desktop mais completo.

Durante o desenvolvimento deste projeto foram praticados conceitos como gerenciamento de layout com Tkinter, criação e organização de menus, manipulação de arquivos, estruturação básica de aplicações desktop e aplicação de boas práticas iniciais para interfaces gráficas.
